# Weekly Kickoff

Run the complete agent team to kick off a new week of content creation and growth.

## Workflow

Execute agents in this order:

1. **Research Phase** (run in parallel):
   - Invoke `reddit-scout` agent to gather fresh Reddit insights
   - Invoke `trend-analyst` agent for current trends
   - Invoke `seo-strategist` agent for keyword opportunities

2. **Strategy Phase**:
   - Invoke `newsletter-strategist` to update growth strategy
   - Invoke `growth-hacker` for new tactics

3. **Planning Phase**:
   - Invoke `team-coordinator` to create weekly plan
   - Review and prioritize with founder

4. **Output**:
   - Present weekly plan summary
   - List top 3 priorities for the week
   - Identify any founder action items

## Expected Time

Full kickoff takes 10-15 minutes of agent processing.

## Usage

Run at the start of each week (Monday preferred).
